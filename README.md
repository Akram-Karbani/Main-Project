# 💫 About Me:
## 🏡 About AirMyHome<br><br>AirMyHome is a full-stack web application designed to streamline the process of discovering and showcasing vacation rentals. Built with a focus on high performance and clean architecture, the application leverages the **MERN** stack and follows the **MVC (Model-View-Controller)** design pattern to ensure code maintainability.<br><br> ✨ Key Features<br>* **Full CRUD Operations:** Users can seamlessly create, read, update, and delete property listings.<br>* **Interactive Mapping:** Integrated **Mapbox GL JS** with forward geocoding to display precise property locations on interactive maps.<br>* **Cloud Image Hosting:** Utilized **Multer** and **Cloudinary** for reliable, cloud-based uploading and rendering of high-resolution property photos.<br>* **Secure Authentication:** Implemented **Passport.js** for robust user session management, ensuring only authorized owners can edit or delete their listings.<br>* **Dynamic UI:** Features a responsive **Bootstrap 5** interface with dynamic client-side logic, including a "Tax Toggle" for transparent pricing.<br><br>### 🛠️ Built With<br>* **Frontend:** HTML5, CSS3, Bootstrap 5, EJS (Embedded JavaScript)<br>* **Backend:** Node.js, Express.js<br>* **Database:** MongoDB, Mongoose ODM<br>* **Core Dependencies:** Passport.js, Cloudinary, Mapbox SDK, Multer, Joi, Connect-Flash
